ATFeeds is a free WordPress plugin that allows you to display custom Twitter feeds

== Description ==
ATFeeds is a free WordPress plugin that allows you to display custom Twitter feeds using shortcode or widgets, it is also fully compatible with Visual Composer. You can customize the appearance of the feeds as you like!. ATFeeds uses the Twitter API.

= Requirements =
> * PHP 5.3 or higher
> * WordPress 4.4 or higher
> * Visual Composer (recommended)

= Features =
> * Very easy to use, 100% free, no ads, no premium version exists
> * Create any feeds as you want
> * Each feed has his own settings
> * Predefined CSS themes (including "naked" theme for custom styling)
> * Visual Composer support
> * Very lightweight
> * Translation-ready (English and Spanish included)
> * Minimalist design and fully responsive
> * And much more!

== Installation ==
1. Upload the plugin to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
= Is Awesome Twitter Feeds free? =
Yes, of course. This plugin is 100% free. No ads, no premium version exists.

= How can i use the available shortcode without Visual Composer? =
> * [awesome-twitter-feeds id="4" title="My Tweets"]
